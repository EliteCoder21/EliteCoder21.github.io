## Overview

Final project for CSE 474 (Embedded Systems). Built an autonomous car powered by the ESP32 and Arduino Mega that navigates towards a target while avoiding obstacles.

## Features

- **Autonomous Navigation**: Self-driving capability with target-seeking behavior
- **Obstacle Avoidance**: Real-time sensing and path planning around obstacles
- **Beacon Navigation**: Guided navigation using beacon signals
- **Wireless Control**: Remote control capability for manual operation
- **Dual Processor Architecture**: ESP32 for wireless/advanced processing, Arduino Mega for motor control

## Technical Implementation

- **Microcontrollers**: ESP32 + Arduino Mega
- **Languages**: JavaScript (38.4%), HTML (29.5%), CSS (28.6%), C++ (3.5%)
- **RTOS**: FreeRTOS for reliable multitasking
- **Project Structure**:
  - `ESP32/` - ESP32 firmware
  - `Arduino_Mega/` - Motor control logic
  - `Remote_Control/` - Web-based remote control interface

## Key Challenges

- Implemented concurrency with FreeRTOS on ESP32 to ensure multiple tasks (navigation, obstacle detection, wireless communication) could run reliably
- Built responsive indoor navigation system with real-time obstacle detection and avoidance
- Created web-based remote control interface for manual override

This project strengthened my understanding of embedded systems, real-time operating systems, and the challenges of building autonomous robots in resource-constrained environments.
